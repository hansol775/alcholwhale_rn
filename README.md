# 술고래(alcholwhale)
술고래 프로젝트 리액트네이티브 버전입니다.

## 설치방법

`git clone git@github.com:hansol775/alcholwhale_rn.git`

`cd alcholwhale_rn`

`npm install`

`cd ios`

`pod install`

`react-native run-ios or react-native-run-android`
